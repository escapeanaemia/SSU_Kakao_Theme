# SSU_Kakao_Theme


숭실 복숭아 테마1은 숭실대학교 비공식 마스코트인 뽁이(복숭아)를 바탕으로 제작한 카카오톡 ios버전 테마입니다.
숭실대 학생 및 일반인도 해당 테마를 자유롭게 사용/변형이 가능합니다.

비공식 마스코트 뽁이는 숭실대 학우에 한하여 자유롭게 2차 변형 및 사용이 가능합니다.
